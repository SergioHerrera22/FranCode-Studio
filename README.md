# Landing page diseño conversión

## Deploy en Netlify

Si Netlify muestra `sh: 1: vite: Permission denied`, este proyecto ya incluye una mitigación en el script de build para ejecutar Vite con Node directamente en lugar del wrapper de `node_modules/.bin`.

Si el sitio ya había fallado antes, conviene lanzar un redeploy con caché limpia en Netlify para regenerar dependencias desde cero.

This is a code bundle for Landing page diseño conversión. The original project is available at https://www.figma.com/design/yYqXWoGiVU2JQGGXwjsgSL/Landing-page-dise%C3%B1o-conversi%C3%B3n.

## Running the code

Run `npm i` to install the dependencies.

Run `npm run dev` to start the development server.
